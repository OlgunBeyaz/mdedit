# Willkommen zu mdedit

Dies ist ein **einfacher Markdown-Editor** geschrieben in C++ mit Qt6.

## Features

- ✅ Neue Dateien erstellen
- ✅ Markdown-Dateien öffnen
- ✅ Speichern und "Speichern als"
- ✅ **Live-Preview** mit Styling
- ✅ Schlichte, benutzerfreundliche GUI

## Markdown-Beispiele

### Code-Blöcke

```cpp
#include <iostream>
int main() {
    std::cout << "Hello Qt6!" << std::endl;
    return 0;
}
```

### Listen

1. Erstelle eine neue Datei
2. Schreibe Markdown-Text
3. Speichere die Datei
4. Betrachte die Live-Preview

### Textformatierung

- **Fett**
- *Kursiv*
- `Code`
